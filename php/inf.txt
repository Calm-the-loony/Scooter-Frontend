Пояснение:
Создание токена: Используется bin2hex(random_bytes(50)), чтобы создать случайный токен для восстановления пароля. Этот токен отправляется пользователю на почту.

Сохранение токена в базе данных: Токен вместе с почтой и сроком действия (30 минут) сохраняется в таблице password_resets. Если запись с таким email уже существует, она обновляется.

Отправка письма: Письмо отправляется с использованием PHPMailer. Оно содержит ссылку на страницу восстановления пароля, которая включает токен.

Настройки SMTP: Замените 'smtp.yourdomain.com', 'your-email@yourdomain.com' и 'your-email-password' на данные вашего SMTP сервера.

Обработка ошибок: Если отправка письма не удалась, в ответе будет ошибка с описанием.

Теперь, если пользователь забудет пароль, он сможет восстановить его, следуя ссылке из письма.



1. Установка Composer
Если у вас ещё не установлен Composer, инструмент для управления зависимостями PHP, вам нужно его установить.

Загрузите Composer:

Для Windows скачайте установщик с официального сайта.
Для Linux и macOS выполните следующие команды в терминале:

php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"
php -r "if (hash_file('SHA384', 'composer-setup.php') === 'эТОТ_HАШ') { echo 'Installer verified'; } else { echo 'Installer corrupt'; unlink('composer-setup.php'); } echo PHP_EOL;"
php composer-setup.php
php -r "unlink('composer-setup.php');"
sudo mv composer.phar /usr/local/bin/composer
Проверьте установку: Откройте командную строку (терминал) и выполните команду:

composer -V
Если всё правильно установлено, вы увидите версию Composer.

2. Установка PHPMailer
Теперь, когда Composer установлен, можно установить PHPMailer:

Перейдите в директорию проекта: В терминале или командной строке перейдите в корневую директорию вашего проекта:

cd /путь/к/вашему/проекту
Установите PHPMailer: Выполните следующую команду, чтобы установить PHPMailer через Composer:

composer require phpmailer/phpmailer
Эта команда загрузит библиотеку PHPMailer и её зависимости в директорию vendor в вашем проекте.

3. Использование PHPMailer
После установки, чтобы использовать PHPMailer в вашем проекте, подключите автозагрузчик Composer в вашем PHP-коде:


require 'vendor/autoload.php';
Теперь вы можете создавать объекты PHPMailer и отправлять письма. Пример кода с использованием PHPMailer приведён в предыдущем сообщении.

4. Проверка установки
Чтобы убедиться, что всё работает, попробуйте отправить тестовое письмо, используя предоставленный ранее пример кода. Если письмо успешно отправлено, значит, библиотека установлена и настроена правильно.
